# Multi-Source Data ETL Pipeline (Batch Processing)

## Overview
This project demonstrates extracting data from multiple sources (CSV, JSON, XML), transforming it, and loading it into a single CSV file. It simulates a real-world ETL workflow with logging and data transformations.

## Tech Stack
Python | Pandas | XML | CSV | JSON

## Features
- Extracts data from CSV, JSON, and XML files
- Transforms height from inches to meters and weight from pounds to kilograms
- Loads transformed data into a single CSV file
- Logs progress and errors
- Modular ETL pipeline with clear structure

## How to Run
1. Install dependencies: `pip install -r requirements.txt`
2. Place CSV, JSON, and XML files in the same folder
3. Run the script: `python etl_pipeline.py`
4. Output CSV and log file will be generated

## Outcome
This project demonstrates a modular ETL pipeline for multiple data sources, including extraction, transformation, and loading — foundational skills for a data engineer role.
